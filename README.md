# The ZSMC Co.

The ZSMC Co. is an independent research initiative focused on the development of safer, lower-cost, and more sustainable zinc-based electrochemical energy storage systems.

This repository contains the source code for the official ZSMC website.

## About

The project serves as the public-facing platform for ZSMC research activities, publications, technical articles, and updates related to zinc-based battery technology and electrochemical storage research.

## Features
- Modern responsive design
- Accessibility-focused implementation
- Performance optimized
- SEO-friendly structure
- Lightweight and fast-loading pages
- Development

## Clone the repository:

git clone https://github.com/The-ZSMC-Co/website-code.git

Open index.html directly in your browser or serve the project using your preferred local web server.
